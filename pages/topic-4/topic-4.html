Computer Hardware

Internal Hardware

Installed on a motherboard, the primary circuit board, and located inside the computer's casing.

External Hardware

Connected to the computer using a USB cable or similar and is located outside the computer’s casing.

Input Hardware

Allows users and computers to interact so that the system can respond, process data, and produce output.

Keyboard

  • enables users to enter data, commands, and text into a computer
  • consists of function keys, letters, numbers, and symbols
  • consists of small, tenkeyless (TKL), and full-sized layouts

Mouse

  • allows users to select, click, and move items on the screen by moving the cursor
  • features a scroll wheel in the middle, as well as buttons on the left and right for accessing and selecting menus

Output Hardware

Gives the user feedback and enables them to access and use the computer's processed data.

Monitor

  • different panel types such as LCD, VN, IPS, and OLED
  • accessible in a broad range of sizes, features, capacities, and costs
  • displays user interfaces, text, photos, and videos to enable interactions with software programs

Printer

  • converts digital documents or images into physical copies onto paper
  • accommodates for different paper sizes such as A4, Letter, and larger formats
  • utilizes toner or laser printers or ink for inkjet printers cartridges

Process Hardware

The ability of a computer to process input data and provide an output.

CPU

  • Central Processing Unit
  • serves as the brain of the computer
  • manages the data flow to and from memory and I/O devices
  • processes instructions and performs arithmetic and logical operations

Graphics Card

  • contains a GPU that is responsible for rendering images, videos, and animations
  • contains video memory that stores image data for quick access by the GPU
  • includes video ports for connecting monitors

Motherboard

  • acts as "Command Central" to manage the system's operations
  • connects all the other components of a computer together
  • includes the chipset, CPU socket, VRM, memory slots, expansion bus slots, and integrated functions

Storage Hardware

Stores a computer’s data either temporarily or long-term for future retrieval and access.

RAM

  • Random Access Memory
  • stores data and software instructions
  • known as volatile memory because it needs a power source to maintain the stored data

Hard Drive

  • can either be a Hard Disk drive (HDD) or Solid State Drive (SSD)
  • serves as the main storage device on any PC
  • data written in the hard drive remains stored until the user deletes it
