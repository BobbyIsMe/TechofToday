What are Number Systems

Number Systems

The number system is a system to write, represent, and express a numerical value that appears in calculations and measures of various sizes.

Within the architecture of a computer system, number systems are the method used to represent numbers.

Quantities and symbols are related in a number system.

Types of Number Systems

Decimal Number System

Decimal numbers have a base of 10, using digits from 0 to 9.

Each digit's position represents a power of 10.

One or more digits can be used to construct a number.

To distinguish between a decimal number's fractional and integral parts, the decimal point is used.

Significance of Decimal Number System

Counting Numbers

People most frequently count in "decimal" or "base-10" numbers in daily life.

Arithmetic Operation

People do arithmetic operations using the decimal number system.

Computer Interaction

Users do human-computer interaction, precise financial calculations, and efficient data representation.

Binary Number System

Binary numbers have a base of 2, using only two digits from 0 to 1.

Each digit's position represents a power of 2.

The smallest unit of binary information kept in a computer system is called a binary digit, or a bit.

Significance of Binary Number System

Computer Operation

Binary numbers represent the electrical signals which a computer hardware operates with. 1 refers to ‘on’, and 0 refers to ‘off’.

Data Storage

Computer memory stores and modifies data in bit sequences using 0s and 1s.

Boolean Logic

Binary number system represents true as 1 and false as 0 values.

Octal Number System

Octal numbers have a base of 8, using digits from 0 to 7.

Each digit's position represents a power of 8.

Binary coded octal numbers are another name for octal numbers represented in binary.

Significance of Octal Number System

Binary Readability

Long binary numbers were frequently represented using octal for human readability.

Binary Conversion

Three binary digits exactly match an octal digit while a decimal requires extra steps to convert into binary digits.

Memory Representation

Binary numbers could be represented more compactly using octal number system in early computer systems.

Hexadecimal Number System

Hexadecimal numbers have a base of 16, using symbols from 0 to 9 and from A to F.
(A = 10, B = 11, C = 12, D = 13, E = 14, F = 15)

Each digit's position represents a power of 16.

Four binary bits are used to represent a hexadecimal digit.

Significance of Hexadecimal Number System

Compact Binary Form

Hexadecimal provides a simplified way of expressing large binary numbers.

Binary Conversion

Each group of four binary digits corresponds to one hexadecimal digit.

Memory Addressing

Modern computer systems use hexadecimal to represent 32-bit memory addresses.
